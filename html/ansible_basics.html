variable precedence (least to most)

  1. role defaults
  2. inventory vars
  3. inventory group_vars
  4. inventory host_vars
  5. playbook group_vars
  6. playbook host_vars
  7. host facts
  8. registered vars

variable precedence (cont)

  1. set_facts
  2. play vars
  3. play vars_prompt
  4. play vars_files
  5. role and include vars
  6. block vars (only for tasks in block)
  7. task vars (only for the task)
  8. extra vars (always win precedence)

reboot


- name: reboot
  command: /usr/sbin/shutdown -r +1 "get off"
  ignore_errors: true
  async: 0
  poll: 0
  sudo: true

- name: wait for ssh port to come down
  local_action: wait_for port=22 host={{ ansible_host }} state=stopped

- name: wait for ssh port to come back up
  local_action: wait_for port=22 host={{ ansible_host }} state=started
